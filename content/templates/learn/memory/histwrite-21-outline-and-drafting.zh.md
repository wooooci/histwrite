# Histwrite-21 大纲 → 分配材料 → 写作连贯（避免机械分段）

> 核心：论文重在“论”。标题只是标路，不是替代论证。

## 1) 大纲不是“标题树”，是“论证链”

每个部分要能回答：
- 这一部分要证明什么？
- 用哪些证据类型证明？
- 预期反驳是什么？怎么处理？
- 如何回扣全文的核心问题？

建议写成段落级推进：
- 判断句 → 证据链 → 解释机制 → 转场。

## 2) 材料分配：先把材料“放对位置”，再动笔

- 用一个材料分配表把证据卡映射到大纲段落：
  - 证据卡ID / 材料类型（一手/二手/民调）
  - 可支撑的判断
  - 适合落在哪一段
  - 需要与哪几份材料并用

- 一段最好组织 2–4 份材料共同支撑同一判断；
  避免“单份材料 + 小段解读”循环往复。

## 3) 连贯性策略：分段写也要像一口气写完

当分段写作不可避免时，用“段落摘要交接”保证连贯：
- 每写完一个部分，输出：
  1) 本部分结论（3–5 条）
  2) 尚未解决的疑问（2–4 条）
  3) 下一部分开头应如何承接（1–2 句建议起笔）

在下一部分写作时，把这份交接摘要作为上文语境，而不是把全文复制进来。

## 4) 学术史段写法（只用二手）

- 功能：交代谱系 → 分歧 → 缺口 → 引出你的切口。
- 禁止把一手史料写成学术史证据。
- 二手文献要“对话化”使用：
  - 哪位作者如何解释？
  - 解释之间在哪些点冲突？
  - 这些解释为何不足以回答你的问题？

## 5) 正文写法：史料只做支点，不做堆砌

- 史料是“论证的证据”，不是“展示材料多”的装饰。
- 史料多不等于论述少：每次引入史料，必须写清它在机制解释中承担哪一环。
- 允许材料密度高，但要用解释把材料串成一条线。

## 6) 数量指标（默认基线）

- 总字数：12000–15000
- 一手史料：≥35（必须服务论证）
- 二手文献：≥20（学术史与关键论证需足量）

## 7) 文风与句法（只提醒，不重复展开）

- 连接词可有但不能依赖；句式模板必须打散；避免冒号块、并列串、引号造概念、黑话与“xx性”。
- “其”可以出现；仅在造成读者回溯、歧义或形成模板腔（如“其+抽象名词”密集）时，改为全称/缩写或重写句子结构。不要把“清除其”当作去 AI 的主抓手。
